# Job Hunting

## 使い方

- @companies.md を中心としてメガベンチャー・有名スタートアップ・外資ITのソフトウェアエンジニア/SRE/FDEポジションを探します
- リージョンは東京がメインですが、関西・福岡・その他Asiaでも探してください。
  検索方法は以下の2種類です

1. 特定の企業のホームページ・就活サイト・LinkedInなどを探索し、ソフトウェアエンジニアリングに関する新規求人があるかどうかをチェックする

2. @参考サイト.md を探索し、新規ポジションがあるかをチェックする

- 新しく取得した求人は以下の情報を添えて、daily-lifeリポジトリのissueの中の「【就活】サマーインターン応募」「【就活】サマーインターン以外の選考」で新しいコメントとして
  追加してください
- 追加する前に収集した企業をリストにしてユーザーに提示してください
- 2026年に開催していて、28卒も対象に入ると判断できる場合に限ります

<必要な情報>

- 企業名
- ポジション名
- ポジションの概要
- 応募期限
- 参考リンク

### 検索する際の注意事項

#### ステップ1: 検索前にissueの全内容を確認する（必須）

まず以下の2コマンドを実行し、issueに登録済みの企業を**全て**洗い出す：

```bash
gh issue view 40 --repo Yuto729/daily-life          # body（本文）に企業名・チェックリストあり
gh issue view 40 --repo Yuto729/daily-life --comments  # コメントにも登録済み企業あり
gh issue view 43 --repo Yuto729/daily-life
gh issue view 43 --repo Yuto729/daily-life --comments
```

⚠️ **`--comments`フラグだけではissueのbody（本文）が取得できない。**
必ず`--comments`なしでbodyを、`--comments`ありでコメントを、両方確認すること。

取得した内容から「登録済み企業リスト」を作成し、@companies.md と照合してから検索開始する。

#### ステップ2: 検索対象企業の絞り込み

- 登録済み企業（issue body + コメント両方に記載された企業）は原則スキップ
- ただし同じ企業でもサマーインターン・通年インターン・本選考と異なるポジションが時期違いでオープンする可能性があります
  - 7月あたりまではインターンのポジションが検索済みであれば、検索は様子見
  - 8月以降の本選考が始まる時期においては, インターンの情報がすでに出ていても本選考の情報が出るまでは検索を続けてください

#### ステップ3: 検索開始

未登録の企業を対象に検索する。

- 企業リストや参考リンクは随時更新します
